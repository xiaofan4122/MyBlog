# 驱动系统测量点

位于车身右侧的两个孔，可以用万用表测现在驱动系统的电压

:::note

如果你在下高压之后一直测量，可以看到电压会快速下降，这是[放电回路](./放电盒子#放电回路)正常工作的结果。

:::
